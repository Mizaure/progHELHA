Untitled :: Model :: Inscrire un lecteur :: (Inscrire un lecteur→Enregistre des emprunts)

Description

Flux général

  • Un lecteur se présente avec des exemplaires auprès de la bibliothécaire
  • Le bibliothécaire idientifie le lecteur grâce à sa carte d'identité
  • Si le lecteur n'est pas identifié, elle a la possibilité de l'inscrire interactivement.
  • Pour chaque exemplaire, le système vérifie s'il n'est pas déjà emprunté.
  • Si c'est le cas, une erreur est affichée et l'exemplaire n'est pas encodé.
  • Une fois tous les exemplaires encodés. le bibliothécaire clot l'encodage avec ce lecteur et toutes les données sont enregistrées dans le système.
